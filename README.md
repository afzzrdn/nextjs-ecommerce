# 📘 NextJS E-commerce

This is a simple ecommerce using NextJS

---

## 🚀 Feature

- 📦 CRUD Data
- 📊 Dashboard Admin
- 🌐 API RESTful
- 💳️ Stripe Payment Gateway

---

## 🛠️ Tech Stack

- **Frontend**: Next.js, Tailwind CSS
- **Backend**:  Next.js
- **Database**: Sanity.io
- **Deployment**: Vercel

---
